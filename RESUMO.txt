🔌 API ENDPOINTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

JOGADORES:
  GET    /api/players              Lista todos (ordenado por rating)
  POST   /api/players              Criar novo
  GET    /api/players/:id          Obter específico
  PUT    /api/players/:id          Atualizar nome
  DELETE /api/players/:id          Remover

PARTIDAS:
  GET    /api/matches              Lista histórico
  POST   /api/matches              Registrar nova
  GET    /api/matches/:id          Obter específica
  DELETE /api/matches/:id          Reverter (recalcula ratings)

SAÚDE:
  GET    /api/health               Status do servidor

📊 BANCO DE DADOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TABELA: players
├─ id                (Primary Key - UUID)
├─ name              (Unique, Max 30 chars)
├─ rating            (Default: 1000)
├─ games, wins, losses
└─ created_at, updated_at

TABELA: matches
├─ id                (Primary Key - UUID)
├─ team_a_p1, team_a_p2  (Foreign Keys)
├─ team_b_p1, team_b_p2  (Foreign Keys)
├─ winner            ('A' ou 'B')
└─ played_at

TABELA: match_deltas
├─ id                (Primary Key - UUID)
├─ match_id, player_id   (Foreign Keys)
├─ delta_rating      (Mudança de rating)
└─ rating_after      (Rating após partida)

❌ "npm não é reconhecido"
✅ Instale Node.js de nodejs.org e reinicie terminal

❌ Porta 3000 em uso

❌ Erro ao criar pasta data/
✅ mkdir data (cria pasta automaticamente)

❌ Banco corrompido
✅ Delete data/bisca.db e reinicie (cria novo)

❌ API não responde
✅ Verifique se servidor está rodando (deve ver logs)

Leia QUICKSTART.md para mais soluções.
